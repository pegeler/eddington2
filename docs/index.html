Eddington Number Online Calculator

This page takes ride mileages and will return the Eddington number for cycling. There are two ways to calculate your Eddington number:

  • Select a .txt or .dat file.
  • Enter your rides in the text box and click "Calculate!"

Both methods allow your distance entries to be either newline separated (e.g., one mileage per line), comma separated, or a combination of both.
