Eddington Number Online Calculator

This page takes daily ride mileages and will return the Eddington number for cycling. There are two ways to calculate your Eddington number:

  • Enter your rides in the text box and click "Calculate!"
  • Load a .txt or .dat file.

Both methods allow your distance entries to be either newline separated (e.g., one mileage per line), comma separated, or a combination of both.

Note that this site can also be used to compute an author's h-index.
